# ForgeFin Page Generator

Always apply:

* forgefin-ui-principles
* forgefin-theme-system
* forgefin-layout-patterns
* forgefin-financial-workflows

---

# Output Requirements

Generate:

1. Page Purpose
2. Layout Pattern
3. Component Tree
4. Leptos Structure
5. Tailwind Classes
6. Theme Compatibility Notes

---

# Page Checklist

Every generated page must include:

* Toolbar
* Search
* Filters
* Main Content Area

Where applicable:

* Tables
* Forms
* KPI Cards
* Export Actions

---

# Completion Standard

Generated UI must:

* Follow ForgeFin design principles
* Support future theme switching
* Follow accounting workflows
* Be desktop-first
* Be information-dense
