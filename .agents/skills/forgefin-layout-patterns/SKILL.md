# ForgeFin Layout Patterns

## Primary Layout

Preferred application shell:

Header + Sidebar + Workspace

Sidebar:

* Width: 240px
* Collapsed: 72px

Header:

* Height: 56px

Workspace:

* Fill remaining area

---

# List Page Pattern

Used for:

* Customers
* Vendors
* Accounts
* Inventory
* Chart of Accounts

Structure:

Toolbar
↓
Filter Area
↓
Data Table
↓
Pagination

---

# Master Detail Pattern

Used for:

* Vouchers
* Purchase Orders
* Payments
* Journal Entries

Structure:

List Panel
↓
Detail Panel

---

# Dashboard Pattern

Top:

KPI Cards

Middle:

Charts

Bottom:

Pending Tasks
Alerts
Recent Activity

# Report Pattern

Structure:

Filter Panel
↓
Toolbar
↓
Report Preview
↓
Export Actions

# Form Pattern

Maximum width:

1200px

Group fields into sections.

Avoid long single-column forms.
