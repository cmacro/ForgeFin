# ForgeFin Design System

Product:
ForgeFin

Domain:
Financial Management Software

Target Users:
- Accountant
- Finance Manager
- CFO
- Auditor

Design Style:
- Enterprise ERP
- SAP Fiori quality
- Ant Design Pro information density
- Professional desktop application
- Data-first design

- Neutral gray palette
- Zinc / Slate colors
- Minimal shadows
- Subtle borders
- Dense tables
- Compact forms

Layout:
- Left Sidebar
- Top Header
- Content Workspace
- Multi-level Navigation

- Professional business appearance

- Marketing website layouts
- Hero sections
- Pricing cards
- Testimonials
- Gradient backgrounds
- Glassmorphism
- Oversized border radius
- Large empty spaces

Radius:
- rounded-md only

Spacing:
- 8px grid system

Typography:
- Data readability first
- Clear hierarchy
- Compact enterprise style
