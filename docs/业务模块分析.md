## 十大核心业务模块

### 1. 项目收支核算

每个项目独立 Sheet，按凭证号逐笔记录收入和支出：

- **收入侧**：工程款收入、现金交入、设计收入
- **支出侧**：材料费、管理费、税金、劳务款、电汇手续费、日常费用支出、废旧物资回购支出、保证金
- **核算方式**：`收入合计 - 支出合计 = 项目利润`

> 示例结构：
> ```
> 日期 | 凭证号 | 内容摘要 | 现金交入 | 工程款收入 | 收入合计
> 材料费 | 管理费 | 税金 | 劳务款 | 电汇手续费 | 日常费用
> 废旧物资回购 | 保证金 | 支出合计 | 付款方式 | 交易单位 | 领款人 | 发票情况
> ```

### 2. 税务管理

各项目单独预缴并记录以下税种：

- 增值税（一般纳税人 / 小规模预缴）
- 城建税
- 教育费附加
- 地方教育附加
- 企业所得税预缴
- 其他税费（印花税等）

典型摘要：`"XXX项目预缴税费：增值税X元，城建税X元，教育附加X元，地方教育附加X元，所得税X元，合计X元"`。

### 3. 材料采购管理

- 材料费明细记录（供应商名称、金额、发票类型）
- 劳务合同与材料合同常绑定记录（如班组合同中材料与劳务分开列示）
- 材料余款跟踪

### 4. 劳务分包管理

- 劳务款支付（班组结算、工资表模式）
- 分包单位管理及银行账户记录
- 合同金额、已付金额、未付余款跟踪

### 5. 保证金与押金管理

- 投标保证金
- 履约保证金
- 装修押金
- 押金退还跟踪

### 6. 费用报销

涵盖以下费用类型：

- 差旅费：油费、过路费、住宿费、餐补、机票、火车票
- 办公费：图文打印、快递费
- 房租水电
- 招待费
- 报销人、发票类型（专票/普票）跟踪

### 7. 收款与付款管理

- **付款账户**：不同银行账户（如企业基本户、一般户）
- **交易单位**：各主体公司
- **银行信息**：开户行名称、银行账号完整记录
- **批量付款**：通过 `=$P4` 列绝对引用实现批量继承

### 8. 发票管理

- 发票情况字段记录：专票 / 普票、金额
- 开票状态：已收 / 后补
- 与付款记录关联

### 9. 废旧物资处置

- 废旧物资回购收入 / 支出
- 涉及物资类型：旧电缆、旧设备、旧空调等

### 10. 汇总报表

按年份单独 Sheet 汇总，结构如下：

| 字段 | 说明 |
|------|------|
| 序号 | 项目流水号 |
| 项目编号 | 内部编码 (如 P001–P200) |
| 会签日期 | 合同签署日期 |
| 工程进度 | 施工中 / 已完工等结算 / 已结算款未齐 / 工程结束 / 作废 |
| 项目名称 | 工程全称 |
| 合同额 | 合同签订金额 |
| 结算价 | 最终结算金额 |
| 各年收款 | 按年份列示已收款金额 |
| 收入/支出/利润 | 汇总统计 |

## 跨 Sheet 引用与导航机制

- **项目 Sheet 跳转**：汇总表中 `='汇总表2022'!A1` 作为导航锚点
- **利润汇总**：`='P001'!O82+'P001'!P82` 从项目 Sheet 取数
- **付款方式继承**：`=$P4` 列绝对引用实现批量填充

## 涉及的会计职能模块

| 职能 | 说明 |
|------|------|
| 收入确认 | 按项目、按期间确认工程款收入 |
| 成本归集 | 料（材料费）/ 工（劳务费）/ 费（日常费用）/ 税（税金）分开归集 |
| 税金预缴与申报 | 各项目单独预缴并汇总 |
| 往来款管理 | 应收工程款、应付材料款/劳务款、保证金 |
| 项目利润核算 | 单项目收入 - 成本 = 利润 |
| 资金流水 | 类银行日记账格式，每日记录收支 |

## 数据结构说明（列定义）

| 列 | 含义 | 数据类型 |
|----|------|----------|
| 日期 | 业务发生日期 | 日期 |
| 凭证号 | 记账凭证编号 | 文本 |
| 内容 | 业务摘要描述 | 文本 |
| 现金交入 | 现金收款 | 数值 |
| 工程款收入 | 银行转账工程款 | 数值 |
| 收入合计 | 公式计算 = 前两列之和 | 公式 |
| 材料费 | 材料采购支出 | 数值 |
| 管理费 | 管理费支出 | 数值 |
| 税金 | 税务支出 | 数值 |
| 劳务款 | 劳务分包支出 | 数值 |
| 电汇手续费 | 银行手续费 | 数值 |
| 日常费用支出 | 报销等日常费用 | 数值 |
| 废旧物资回购支出 | 废旧物资回收成本 | 数值 |
| 保证金 | 各类保证金/押金 | 数值 |
| 支出合计 | 公式计算 = 各支出列之和 | 公式 |
| 付款方式 | 付款银行账户 | 文本 |
| 交易单位 | 交易主体公司 | 文本 |
| 领款人 | 收款方信息（含银行账号） | 文本 |
| 发票情况 | 发票类型与状态 | 文本 |
