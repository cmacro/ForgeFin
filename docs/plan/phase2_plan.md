# Phase 2: 自动化和导入（效率提升）

## 目标
提升业务效率，降低手工录入成本。

## 任务列表

- **2.1 Excel 导入凭证**
  - 解析 `.xlsx`，自动映射列到凭证字段。
  - 支持批量导入、错误行回报。

- **2.2 银行流水导入**
  - CSV/Excel 导入，自动匹配科目。
  - 支持自定义映射规则，导入前预览。

- **2.3 自动生成凭证**
  - 根据业务模板（如采购入库 → 借库存 / 贷应付）自动产生凭证。
  - 支持多模板管理、模板参数化。

- **2.4 发票 OCR**
  - 图片上传 → 识别日期、金额、税率、供应商、发票号。
  - 识别后可直接生成凭证草稿。
